# KUI-Mobile
 基于React开发的移动端组件库

 [Demo](https://kotenei.github.io/KUI-Mobile/dist/index.html#/)

## 安装
```bash
npm install kui-mobile --save
```

## 示例
```jsx

import { Button } from "kui-mobile";

ReactDOM.render(<Button>default</Button>, mountNode);

```

### 引入样式

```jsx
import "kui-mobile/lib/kui-mobile.min.css";

```
