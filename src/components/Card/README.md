# API

```jsx
import {Card, CardHeader, CardBody, CardFooter } from 'kui-mobile';
```

## Card

### Props

| 名称 | 类型    | 默认值 | 描述     |
| ---- | ------- | ------ | -------- |
| full | boolean | -      | 是否通栏 |

## CardHeader

### Props

| 名称  | 类型                      | 默认值 | 描述     |
| ----- | ------------------------- | ------ | -------- |
| extra | React.ReactNode \| string | -      | 右则操作区 |


## CardFooter

### Props

| 名称  | 类型                      | 默认值 | 描述     |
| ----- | ------------------------- | ------ | -------- |
| extra | React.ReactNode \| string | -      | 右则操作区 |
